	Let $S$ be some set, and let $\rho$ be some equivalence relation on $S$. 
	\begin{enumerate}[label=(\alph*)]
		\item Define a new relation $\rho$ on S for which $x\rho y$ if and only if $\exists z\in S: x\rho y,\ y\rho x$. Prove that yρz . Prove that σ = ρ.
	\end{enumerate}
(b) Let T be some set, and let f : S → T be a one-to-one function. Define a relation ν on
T whereby, for all a, b ∈ T , aνb if and only if ∃x, y ∈ S such that xρy , f (x) = a, and
f (y) = b. Is it always true that ν must be an equivalence relation? Either prove that
it is, or provide a counterexample.
(c) Let T be some set, and let g : S → T be an onto function. Define a relation μ on T
whereby, for all a, b ∈ T , aμb if and only if ∃x, y ∈ S such that xρy , g(x) = a, and
g(y) = b. Is it always true that μ must be an equivalence relation? Either prove that
it is, or provide a counterexample.
